# TeacherAide 🏫

> AI lesson planning assistant for teachers in under-resourced schools
> Built with Microsoft 365 Copilot + Copilot Studio + Work IQ

## What It Does

TeacherAide is a Microsoft 365 Copilot agent that helps under-resourced
teachers plan lessons, create assessments, and find free teaching
activities — in seconds, not hours.

| Ask TeacherAide | It Returns |
|---|---|
| "Plan a Grade 5 Maths lesson on fractions" | Full 45-min lesson plan cited from NCERT |
| "Create a quiz on photosynthesis" | 10 questions + answer key |
| "Low-cost activity for geography" | 3 free classroom activities |
| "Help for mixed-ability students" | Differentiated lesson from Bloom's Taxonomy |

## How It Works

1. Teacher opens TeacherAide inside Microsoft Teams
2. Types a request OR taps one of 5 starter buttons
3. Agent retrieves grounded answer from curriculum docs via **Work IQ**
4. Returns cited, structured, classroom-ready response in seconds

## Microsoft Work IQ Integration

Knowledge base indexed in SharePoint:
- NCERT Syllabus (Grades 1-10)
- Bloom's Taxonomy Framework
- UNESCO Lesson Plan Guidelines

Every answer is grounded and cites its source document.

## Accessibility Features

- 5 one-tap starter buttons (no typing needed)
- Plain language responses
- Mobile-friendly interface
- Works inside existing Microsoft Teams
- Video demo includes auto-generated captions (WCAG 2.1)
